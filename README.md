# 🕌 Sajdah

A comprehensive Islamic companion web app built with HTML, CSS, and JavaScript. Sajdah is designed to support Muslims in their daily worship and spiritual growth — bringing together a wide range of features under one clean, mobile-friendly interface.

🔗 **Live App:** [sajdah2.web.app](https://sajdah2.web.app)

---

## Features

### 📖 Quran
A full digital Quran reader with a clean and accessible UI for daily reading and reflection.

### 🤲 Duas Library
A curated collection of essential daily Duas, each with Arabic text, transliteration, English translation, and audio recitation — covering everything from entering the masjid to travelling and times of distress.

### 🙏 Salah Guide
A detailed step-by-step guide to performing Salah correctly, complete with Arabic text, transliteration, and audio for each position — from Takbeer to Salaam. Ideal for new Muslims or anyone looking to refine their prayer.

### ⚡ Ayatul Kursi
A dedicated page for Ayatul Kursi with Arabic text, transliteration, and translation for easy daily recitation.

### 🕌 Masjid Finder
A live Masjid Finder built using the Google Maps API and Leaflet.js — plotting nearby mosques on an interactive map using the device's GPS, with a 5km radius filter and custom mosque icons.

### ⏰ Prayer Timings
Accurate daily prayer times to help users stay consistent with their five daily prayers.

### 📿 Hadiths
A curated Hadith section for daily reading and spiritual reflection.

### 🌍 Prophets
Stories and information about the Prophets of Islam for learning and reflection.

### 🤖 AI Assistant
An integrated AI assistant powered by a Node.js backend deployed on Render. Users can ask Islamic questions and receive real-time responses — adding a genuinely interactive and educational dimension to the app.

### 👤 User Profile
A personalised profile section for customising the user experience, with plans to expand into saved content and progress tracking.

### 🕋 Islamic Content
Features Ayatul Kursi with audio, the 99 Names of Allah, Prophets of Islam, a Hadiths section, and a Hajj guide — giving users a holistic resource for learning and reflection.
